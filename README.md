# The Japan COVID-19 Task Force (JCTF) whole blood RNA-seq analysis


Welcome to the github page for the [The whole blood transcriptional regulation landscape in 465 COVID-19 infected samples from Japan COVID-19 Task Force. Nat Commun nn, nnnn (2022). paper](https://github.com/FinucaneLab/Expression_Modifier_Score), where we report on a thorough analysis of whole blood RNA-seq data from 465 genotyped samples from the Japan COVID-19 Task Force, including 359 severe and 106 non-severe COVID-19 cases.

The scripts used in the aper are stored in the code directory.
